NFC Plugin Demo

The NFC plugin allows you to read and write NFC tags. You can also beam to and receive from other NFC enabled devices.

You can do many cool things with NFC, this demo will showcase a few.

Custom plugins don't work in the simulator and the AppBuilder companion app, so inside Telerik AppBuilder choose Run - Build - Android/Windows Phone - App package.

Check availability

For Android only. Error callback message is NO_NFC (no hardware support) or NFC_DISABLED (supported but disabled).
